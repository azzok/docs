// DOCUMENTATION · v2.4.1

Build faster with Azzok

A lightweight, zero-dependency JavaScript utility library for modern web applications. Typed, tree-shakeable, and production-ready.

✓ Stable ESM + CJS MIT License TypeScript 3.2 kB gzipped

init(options)

Creates and returns a configured Azzok instance. Must be called before using stateful methods.

TYPESCRIPT 
function init(options?: InitOptions): KitInstance

Parameters

Parameter Type Required Description
debug boolean optional Enables verbose console logging. Default: false
timeout number optional Request timeout in milliseconds. Default: 5000
retries number optional Number of retry attempts on failure. Default: 3
onError Function optional Global error callback (err: Error) => void

Methods

debounce(fn, delay)

Returns a debounced version of the provided function, delaying invocation until after delay milliseconds have elapsed since the last call.

TYPESCRIPT 
debounce<T extends (...args: any[]) => any>(fn: T, delay: number): T

deepMerge(...objects)

Recursively merges two or more plain objects. Arrays are replaced, not concatenated. Returns a new object.

JAVASCRIPT 
const result = deepMerge(
  { a: 1, b: { c: 2 } },
  { b: { d: 3 }, e: 4 }
);
// → { a: 1, b: { c: 2, d: 3 }, e: 4 }
Note: deepMerge only handles plain objects. Passing class instances or arrays at the top level produces undefined behavior.

throttle(fn, interval)

Limits fn to fire at most once per interval ms. Useful for scroll and mousemove handlers.

once(fn)

Wraps a function so it only executes the first time it is called. All subsequent calls return the original result.

Options Reference

All options can be overridden per-call when using stateful instance methods.

Option Type Default Description
debug boolean false Enable debug logging to the console
timeout number 5000 Milliseconds before a request times out
retries number 3 Retry count for transient failures
cache boolean true Cache computed results in memory
strict boolean false Throw on invalid arguments instead of silently failing

Examples

Handling async with retry

JAVASCRIPT 
import { init } from 'azzok';

const kit = init({ retries: 3, debug: true });

const data = await kit.fetch('https://api.example.com/data');
console.log(data);

Combining utilities

JAVASCRIPT 
import { debounce, once } from 'azzok';

const setup = once(() => {
  console.log('initialized');
});

const onInput = debounce((e) => {
  setup(); // only runs once
  console.log(e.target.value);
}, 300);
